# books-slim
Маленький учебный сайт.

Реализовано:
1. На главной странице отображаются записи из Базы Данных, совмещены две таблицы в одну.
2. Авторизация и Регистрация пользователя.
3. В случае, если пользователь авторизуется под алминистратором, выполяется редирект в админку.
4. Полный CRUD для книг: админ может добавлять, смотреть, редактировать и удалять записи с книгами.
5. Показывается история книги и возвращена ли она в "библиотеку" или её ещё нет в наличии.
